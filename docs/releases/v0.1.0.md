# Night Voyager v0.1.0

## Classification

v0.1.0 is a **local synthetic portfolio release**. It packages reproducible engineering evidence for an advisor-to-family decision workflow; it is not a production deployment, production-tenancy claim, or admissions service.

## Outcome

The connected `/demo` follows one synthetic Case from an advisor planning task through deterministic evidence gates, advisor review, explicit family confirmation, and a persisted receipt and timeline. The browser consumes role-scoped projections through an explicit transport-only BFF while PostgreSQL remains the system of record.

## Included proof

- Four forced-RLS migrations and runtime-equivalent database roles.
- Durable `AgentTask` execution, bounded leases, generation fencing, retry/cancel behavior, and authorized SSE replay.
- Deterministic planning, evidence provenance, advisor review, and family-decision authority.
- A real Chromium advisor-to-family walkthrough at 1440, 768, and 390 pixel viewports.
- An optional, isolated M4B read-only MKE compatibility proof whose output remains `UNTRUSTED_CANDIDATE` and disconnected from the product path.

## Verification

The release candidate is checked with the repository's Python, frontend, PostgreSQL/RLS, installed-wheel, Compose, and Chromium gates. Reproduce the evidence with the [v0.1.0 verification guide](../how-to/verify-v0.1.0-release.md).

## Artifacts

The only published artifacts for v0.1.0 are the GitHub-generated source archives. No separately built wheel, container image, or binary is a release artifact.

## Public copy rules

- Pull request titles use concise English Conventional Commit style; descriptions use Simplified Chinese with English headings unless an external collaboration requires English.
- The GitHub Release title and notes identify v0.1.0 as a local synthetic portfolio release and link reproducible repository evidence.
- Release copy must not claim production deployment, real users, admissions outcomes, SLA, availability, measured savings, or business impact.

## Limits

All Cases and actors are synthetic. Remote providers, messaging, production tenancy, deployment, and real student data are outside this release. The evidence demonstrates local behavior only and does not predict admissions outcomes.
