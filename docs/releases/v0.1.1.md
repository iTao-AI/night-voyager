# Night Voyager v0.1.1

> Release type: `local synthetic portfolio release`

## Summary

Night Voyager `v0.1.1` 在既有 local synthetic advisor-to-family workflow 上新增 deterministic offline governed DRA closure。DRA candidate import、人工 verification/promotion 与 mixed PlanningRun 通过 Night Voyager 自有 authority contracts 接入；外部研究结果不能自行提升 authority。现有 `/demo` 继续作为 local synthetic walkthrough，不接入 live DRA。

## Completion

- 新增 deterministic offline governed DRA candidate import；import 起点固定为 `UNTRUSTED_CANDIDATE`，adapter 或 provider 输出不能自行提升 authority。
- 新增 assigned advisor human verification/promotion，并保持为一个 atomic authority gate；reject 不生成 promoted source pack 或 Evidence。
- 唯一允许的 external mapping 是 `australia_program_fit -> program_fit -> externally_verified`。
- 其他 source/Evidence、cost/FX 与 ranking facts 锁定为 synthetic baseline 的精确副本。
- student/family facts 来自 pinned current Case revision；caller 不能选择 baseline pins、promoted identities 或 authority。
- 新增 governed mixed PlanningRun generation，并通过 existing durable worker 执行；AdvisorReview、DecisionBrief、family decision、DecisionReceipt 与 TimelinePlan 继续复用既有 authority。

## Verification

Release candidate 必须依次通过 `make doctor MODE=dev`、`uv lock --check`、`make dra-check`、`make db-check`、`make check`、`make proof`、`make compose-proof`、`make down`、最终为空的 `docker compose ps --all`、clean-tree `scripts/verify_release.py --tree-mode release`、`git diff --check` 与 public-hygiene review。

Live provider proof was not run；它不属于 required CI、offline proof 或本次 release gate，仍需单独授权。

## Scope

本版本的发布 artifact 是 GitHub-generated source archive。新增能力限定为 deterministic offline governed DRA candidate import、atomic human verification/promotion 与 existing durable worker 上的 governed mixed PlanningRun；live provider proof、live DRA product integration、deployment、hosted SaaS 与 production tenancy 不属于本版本。现有 `/demo` 保持原有 local synthetic walkthrough，不接 live DRA。

## Risk / Impact

本版本证明有界、可复现的 deterministic offline governed DRA closure，不证明 live provider 可用性、production tenancy、真实学生或机构覆盖、录取结果、SLA、性能收益或业务影响。未运行 live provider proof。

## Documentation impact

Bilingual README、documentation index、contributor/security entry、release notes 与 source-archive verification guide 现在以 `v0.1.1` 为 current release；已发布的 `v0.1.0` release notes 与 verification guide 保持不可变历史记录。
