# Contributing

Night Voyager is in bootstrap stage. Keep changes focused, public-neutral, and reproducible without remote credentials.

1. Use Python 3.12.13 and Node.js 24.18.0.
2. Create a short-lived `codex/` branch and preserve unrelated work.
3. Write a failing test before behavioral code.
4. Run `make check` and inspect `git diff --check` before committing.
5. Do not commit `.env`, credentials, private paths, real student data, or unsupported public claims.

Pull requests should state scope, actual verification, documentation impact, and remaining risk.
